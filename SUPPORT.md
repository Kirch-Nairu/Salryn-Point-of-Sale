# Support

For inquiries, setup, pricing, and support:

```text
salryn.info@gmail.com
```

## Support Scope

Support may include:

- installation guidance
- store setup guidance
- backup guidance
- license assistance
- receipt or data location guidance
- troubleshooting daily POS use

Hardware, network cabling, printer setup, barcode scanner setup, data migration, product encoding, and on-site service may be quoted separately depending on the client setup.

## When Asking for Help

Please include:

- store or organization name
- Windows version
- Salryn POS installer version
- short description of the issue or request
- screenshots when useful
