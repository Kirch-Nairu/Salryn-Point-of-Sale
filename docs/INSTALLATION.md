# Installation Guide

## Step 1 — Download

Download the latest Salryn POS setup file from the GitHub Releases page.

## Step 2 — Run the Installer

Run the setup file and follow the setup wizard.

## Step 3 — Open Salryn POS

After installation, open:

```text
Salryn POS
```

from the desktop shortcut.

## Step 4 — Start Store Setup

Set up your store profile, user account, products, and starting inventory.

---

## Important Data Location

Salryn stores local store data outside the install folder.

Do not delete the Salryn data folder unless instructed during support.

It stores local store data such as:

- database
- receipts
- backups
- exports
- logs
- configuration
- license state

---

## Internet

No internet is needed for daily store use.

Internet may be needed for downloading the installer, updates, setup help, or support services.
