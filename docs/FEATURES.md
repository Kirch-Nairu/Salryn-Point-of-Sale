# Salryn POS Features

Salryn POS is built for small-store operations.

---

## Point of Sale

- Process sales from the POS checkout screen
- Accept cash payment
- Save completed sales
- View sales history
- Review receipt details
- Reprint saved receipts
- Start a new sale after checkout

---

## Inventory Tracking

- Create and manage products
- Track item quantities
- Record stock-in
- Record stock adjustments
- Review stock movements
- Monitor low-stock items
- Use barcode/SKU values for product lookup
- Use units of measure such as piece, box, pack, or custom store units

---

## Sales Tracking

- View daily sales
- View weekly sales
- View monthly sales
- Review completed transactions
- Track voided sales separately
- Monitor store performance from the dashboard

---

## Receipts

- Save receipt files
- Show receipt number
- Show cashier name
- Show item lines
- Show subtotal, total, cash, change, and payment method
- Reprint saved receipts

---

## Backup and Records

- Create backups
- Restore from backup
- Export audit records
- Keep local store records organized

---

## Offline-Ready Use

Salryn POS is designed for daily store use without depending on internet connection.

The store can continue using POS checkout, inventory, receipts, and sales history even when internet is unavailable.

---

## Terminal Expansion

Additional Terminal Access allows supported devices on the same local network to access selected Salryn workflows.

Examples:

- Extra cashier station
- Tablet cashier workflow
- Phone or tablet light access
- Manager review access

The main Windows host remains the store's primary Salryn system.
