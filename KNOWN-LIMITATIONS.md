# Current Scope

Salryn POS Standard is focused on local small-store POS operations.

## Included

- POS checkout
- Inventory tracking
- Sales tracking
- Receipt files
- Dashboard
- Audit export
- Backup and restore
- Windows app installation
- Additional terminal access support

## Not Included by Default

- POS hardware
- Printer hardware
- Barcode scanner hardware
- Cash drawer hardware
- BIR accreditation processing
- eSales submission service
- Payment gateway integration
- Multi-branch sync

These can be discussed separately depending on client requirements.
