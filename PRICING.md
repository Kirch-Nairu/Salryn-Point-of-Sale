# Pricing

Salryn POS pricing is for software only.

Hardware, on-site work, product encoding, data migration, network setup, and support services may be quoted separately depending on the client setup.

---

## Salryn Standard

```text
₱20,000 one-time
```

For **1 store / 1 host workstation**.

Includes:

- POS checkout
- Product catalog
- Inventory tracking
- Stock-in and stock adjustment
- Sales history
- Sales dashboard
- Receipt file saving
- Receipt reprint support
- Audit export
- Backup and restore support
- Windows app installation

---

## First 10 Clients Launch Price

```text
₱15,000 one-time
```

Available only for the first 10 client stores.

Includes the same Salryn Standard software package for 1 store / 1 host workstation.

---

## Additional Terminal Access

```text
₱4,000 one-time per terminal
```

For supported extra cashier, tablet, phone, or manager-review access on the same local network.

The Windows host workstation remains the main store system.

---

## Software Only

Prices are for Salryn POS software only.

Not included by default:

- Computer or laptop
- Printer
- Barcode scanner
- Cash drawer
- Router or network hardware
- UPS
- Tablet or phone
- BIR accreditation processing
- eSales submission service
- Payment gateway integration

---

Optional services may be quoted separately:
